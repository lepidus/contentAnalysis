# Plugin Análise de Conteúdo

Este plugin verifica o conteúdo do documento submetido em busca de determinados elementos e metadados da submissão. Em seguida, exibe um relatório dos elementos encontrados na visualização da submissão.

Os conteúdos verificados na análise são:
* Declaração de contribuição dos autores
* Declaração de conflito de interesses
* Declaração de aprovação por comitê de ética
* ORCID dos autores
* Metadados em inglês (título, palavras-chave e resumo)

## Compatibilidade

A última versão deste plugin é compatível com as seguintes aplicações PKP:

* OPS 3.5.0

## Download do Plugin

Para baixar o plugin, acesse a [Página de Versões](https://github.com/lepidus/contentAnalysis/releases) e baixe o pacote tar.gz da versão mais recente compatível com o seu site.

## Dependências de Instalação
Para que o plugin funcione, as seguintes dependências devem estar instaladas no servidor onde a aplicação PKP está sendo executada.

* [poppler-utils](https://poppler.freedesktop.org/)

## Instalação

1. Acesse a área de administração do seu site OPS através do __Painel de Controle__.
2. Navegue até `Configurações` > `Website` > `Plugins` > `Enviar um novo plugin`.
3. Em __Enviar arquivo__, selecione o arquivo __contentAnalysis.tar.gz__.
4. Clique em __Salvar__ e o plugin será instalado no seu site.
